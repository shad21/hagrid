HaGrid

Hagrid est un framework CSS mobile first en cours de développement. Il utilise le préprocesseur SASS, emploie le reset CSS Normalize.css et est entièrement responsif.

Installation

> Prêt à l'emploi:

Framework prêt à l'emploi sans personnalisation possible: Hagrid

> Personnaliser avec compass

Télécharger le framework: Hagrid

Il faut avoir compass & sass d'installer sur sa machine et lancer compass watch pour pouvoir prendre en compte les modifications (si l'on souhaite en faire).
Des variables permmettent de customizer les éléments (boutons, tableaux, typographie, alertes, pagination, fil d'ariane, barre de navigation, etc.). Elles sont accessibles depuis chacun des fichiers _nomDuComposant.scss.

Les composants modifiables sont:

  • _alert.scss
  • _breadcrumb.scss
  • _button.scss
  • _grid.scss
  • _jumbotron.scss
  • _navbar.scss
  • _pagination.scss
  • _table.scss
  • _typo.scss
