Barre de navigation

La barre de navigation emploie le plugin SlickNav écrit en Jquery. Il permet l'affichage du menu sur écran de petite taille.

Installation

Il faut, au préalable, importer la librairie javaScript modernizr.js, JQuery ainsi que jquery.slicknav.min.js. Les fichiers se situent à la racine du framework, dans le dossier js/. Il est préférable de les importer à la fin du document HTML, juste avant la fermeture du </body>. 

<script src="js/modernizr.js"></script>
<script src="js/jquery-1.11.1.min.js"></script>
<script src="SlickNav/jquery.slicknav.min.js"></script>

Pour créer une barre de navigation il faut déclarer un élément <nav> comme suit: 

<nav class="navbar" id="menubar">
    <ul>
        <li><a href="">Home</a></li>
        <li><a href="">Bouton</a></li>
        <li><a href="">Tableaux</a></li>
        <li><a href="">Grille</a></li>
        <li><a href="">Typo</a></li>
        <li><a href="">Alertes</a></li>
        <li>Navigation
            <ul>
                <li><a href="">Navbar</a>/li>
                <li><a href="">Breadcrumb</a></li>
                <li><a href="">Pagination</a></li>
            <u/>
        </li>
        <li><a href="">Jumbotron</a></li>
    </ul>
</nav>

Enfin, il faut initialiser le script comme suit:

<script>
    $(function(){
        $('#menubar').slicknav();
    });
</script>

Exemple

Redimensionner la fenêtre pour que le menu apparaîsse en faut de l'écran.
