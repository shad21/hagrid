Fil d'Ariane

Repère permettant à l'utilisateur de savoir à quel endroit il se trouve sur le site.

Class Description
.breadcrumb Ajoutez cette classe à un élément <ul> afin de créer un fil d'Ariane. Utilisez <a> pour chaque lien du fil d'Ariane.
.current Ajoutez cette classe sur un élément <li> pour définir la page sur laquelle vous êtes actuellement .
.disable Ajoutez cette classe sur un élément <a> pour lui donné un état 'indisponible'.

Exemple

<ul class="breadcrumb">
    <li><a href="#">Accueil</a></li>
    <li><a href="#">Homme</a></li>
    <li class="disable"><a href="#">Costume</a></li>
    <li class="current"><a href="#">Gilet de costume</a></li>
</ul>
